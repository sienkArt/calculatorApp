 ----> Instructions to Run the Program <----


1. If the file is in the same directory as the JAR file:

Open the command line interface (CLI) and navigate to the directory containing the JAR file. Then run the following command:

java -jar calculator.jar filename

Replace filename with the name of your TXT file without the extension e.g. "java -jar calculator.jar test_file"



2. If the file is in a different directory:

Open the command line interface (CLI) and navigate to the directory containing the JAR file. Then run the following command:

java -jar calculator.jar path/to/your/file/filename
Replace path/to/your/file/ with the actual path to your file, and filename with the name of your TXT file without the extension
e.g. "java -jar calculator.jar C:\Users\admin\Desktop\test_files\test_file"


Ensure that you have the necessary access permissions to the file and that the paths provided are correct.
